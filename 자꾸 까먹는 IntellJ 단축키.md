# 으 이클립스랑 헷갈린다!

- try-catch
   - 커서 올려놓고 Ctrl + Alt + T

- 한 줄 삭제
   - Ctrl + Y

- 한 줄 복제
   - Ctrl + D

- 괄호 빠져나오기
   - Tab

- 괄호 완성 후 ; 붙이기, 포멧(if, for, 메서드 등등 (){} 까지 완성하기)
   - Ctrl + Shift + Enter

- 추천
   - Ctrl + Space

- 에러 해결 추천(throws ~, import static 등등)
   - Alt + Enter

- 테스트 클래스 작성
   - 클래스명에 커서 올리고 Ctrl + T
